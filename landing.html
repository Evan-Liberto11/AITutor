How It Works

This system is built using a combination of HTML, CSS, JavaScript, Bootstrap, jQuery, and Ajax to create a fully responsive and visually appealing interface. It consist of five pages:

  • Landing Page: Serves as the 'Home Page'
  • Personal Information Pages: Showcases personal information for each creator
  • AI Tutor Page: Allows users to interact what the Personal Tutor
  • History Page: Provides a detailed summary log of the users prompt and responses

Each page has a responsive navbar to switch tabs

Key Features
  1. AI Tutor:

    Users input questions and specify their experience level. These inputs are processed through API's to retrieve relevant data, and provide personalized responses. All interactions are stored in a database.

  2. History:

    Displays past prompts and AI responses, with advanced filtering and daily summaries.
